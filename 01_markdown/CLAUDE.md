# CLAUDE.md

このファイルは、Claude Code（claude.ai/code）がこのリポジトリで作業する際のガイダンスを提供します。

## プロジェクト概要

Markdown記法を解説する教育コンテンツプロジェクトです。Web開発スキル習得のための学習教材として使用します。

## サンプルファイルの構成

各サンプルファイルは以下の構成で統一：

1. **解説**：記法の説明と用途
2. **記法サンプル**：コードブロック内にMarkdown記法を表示
3. **表示結果**：実際のレンダリング結果
4. **実用例**：実務での使用パターン
5. **実習**：読者が試せる練習問題

## 執筆ガイドライン

- 実用的でコピー＆ペーストできる例を含める
- 生のMarkdown記法と期待される出力を並べて表示
- 基本から応用へ段階的に進める
