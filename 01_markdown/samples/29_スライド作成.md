# スライド作成（Markdown Slides）サンプル

## 解説

Markdownを使ってプレゼンテーションスライドを作成できます。PowerPointを使わずに、テキストベースでスライドを管理できるため、Gitでのバージョン管理も容易です。

| ツール | 特徴 | 環境 |
|--------|------|------|
| Marp | シンプル、VSCode拡張あり | VSCode / CLI |
| Slidev | Vue.js製、インタラクティブ | Node.js |
| reveal.js | 高機能、カスタマイズ性高い | ブラウザ / Node.js |
| Remark | 軽量、ブラウザのみ | ブラウザ |
| Deckset | 美しいデザイン | Mac専用（有料） |

---

## Marp（推奨）

最も手軽に始められるツールです。VSCode拡張機能をインストールするだけで使えます。

### セットアップ

1. VSCodeで拡張機能「Marp for VS Code」をインストール
2. `.md` ファイルを作成
3. フロントマターに `marp: true` を追加
4. プレビューでスライド表示を確認

### 基本構文

```markdown
---
marp: true
---

# スライドタイトル

発表者名
2025年2月

---

## 2枚目のスライド

内容を書く

---

## 3枚目のスライド

- 箇条書き1
- 箇条書き2
```

**ポイント**：`---`（水平線）でスライドを区切ります。

---

## Marpの設定オプション

### フロントマター設定

```markdown
---
marp: true
theme: default
paginate: true
header: 'ヘッダーテキスト'
footer: '© 2025 Your Name'
backgroundColor: #fff
color: #333
---
```

| 設定 | 説明 | 値の例 |
|------|------|--------|
| `marp` | Marp有効化 | `true` |
| `theme` | テーマ | `default`, `gaia`, `uncover` |
| `paginate` | ページ番号表示 | `true`, `false` |
| `header` | ヘッダー | テキスト |
| `footer` | フッター | テキスト |
| `backgroundColor` | 背景色 | `#ffffff`, `black` |
| `size` | スライドサイズ | `16:9`, `4:3` |

## Slidev（開発者向け）

Vue.jsベースの高機能スライドツールです。

### セットアップ

```bash
npm init slidev@latest
```

### 基本構文

```markdown
---
theme: seriph
background: https://example.com/bg.jpg
class: text-center
---

# Slidevへようこそ

開発者のためのプレゼンテーション

---

# コード実行

```ts {monaco-run}
console.log('Hello, Slidev!')
```

---
layout: two-cols
---

# 左カラム

::right::

# 右カラム
```

### Slidevの特徴

- ライブコーディング対応
- アニメーション豊富
- LaTeX数式サポート
- 録画機能内蔵

---

## reveal.js

HTMLベースの高機能プレゼンテーション。

### Markdownでの使用

```html
<section data-markdown>
  <textarea data-template>
    ## スライド1

    内容

    ---

    ## スライド2

    - ポイント1
    - ポイント2
  </textarea>
</section>
```

### 特徴

- 縦横のスライド移動
- 豊富なトランジション
- プラグインシステム
- PDFエクスポート

## エクスポート

### MarpでPDF出力

1. VSCodeでスライドを開く
2. コマンドパレット（`Ctrl+Shift+P`）
3. 「Marp: Export Slide Deck...」を選択
4. PDF、PPTX、HTMLから選択

### CLIでの出力

```bash
# Marp CLIをインストール
npm install -g @marp-team/marp-cli

# PDFに変換
marp slide.md -o slide.pdf

# HTMLに変換
marp slide.md -o slide.html

# PPTXに変換
marp slide.md -o slide.pptx
```
