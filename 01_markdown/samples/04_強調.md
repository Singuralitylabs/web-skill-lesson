# 強調（Emphasis）サンプル

## 解説

テキストを強調する方法は4種類あります。

| 強調の種類 | 記法 | 用途 |
|------------|------|------|
| 太字 | `**text**` | 重要な語句 |
| 斜体 | `*text*` | 英語での強調（日本語では読みにくい） |
| 太字+斜体 | `***text***` | 非常に重要 |
| 打ち消し線 | `~~text~~` | 取り消し、訂正 |

---

## 太字（Bold）

### 記法

```markdown
これは**太字**です。
これは__太字__です（アンダースコアでも可）。
```

### 表示結果

これは**太字**です。
これは__太字__です（アンダースコアでも可）。

---

## 斜体（Italic）

### 記法

```markdown
これは*斜体*です。
これは_斜体_です（アンダースコアでも可）。
```

### 表示結果

これは*斜体*です。
これは_斜体_です（アンダースコアでも可）。

---

## 太字かつ斜体

### 記法

```markdown
これは***太字かつ斜体***です。
これは___太字かつ斜体___です。
```

### 表示結果

これは***太字かつ斜体***です。
これは___太字かつ斜体___です。

---

## 打ち消し線（Strikethrough）

### 記法

```markdown
価格：~~1,000円~~ → 800円
この機能は~~削除予定~~廃止されました。
```

### 表示結果

価格：~~1,000円~~ → 800円
この機能は~~削除予定~~廃止されました。

---

## 組み合わせ例

```markdown
**重要**：この設定は*必ず*確認してください。

~~旧価格：5,000円~~ → **新価格：3,980円**

***緊急のお知らせ***：システムメンテナンスを実施します。
```

### 表示結果

**重要**：この設定は*必ず*確認してください。

~~旧価格：5,000円~~ → **新価格：3,980円**

***緊急のお知らせ***：システムメンテナンスを実施します。

## 生成AIでよく見る使い方

```markdown
**質問**：マークダウンとは何ですか？

**回答**：マークダウンは、テキストに簡単な記号を加えるだけで
書式を表現できる**軽量マークアップ言語**です。

**ポイント**：
- シンプルで覚えやすい
- どこでも使える
- 生成AIの標準出力形式
```

## 実習

以下のテンプレートを使って、商品紹介を書いてみましょう。

```markdown
## 商品名：**〇〇〇**

*人気No.1*の商品です！

**特徴**：
- 軽量
- 高性能
- お手頃価格

~~通常価格：10,000円~~ → **特別価格：7,980円**
```
