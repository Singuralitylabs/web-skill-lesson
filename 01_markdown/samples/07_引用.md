# 引用（Blockquotes）サンプル

## 解説

他の文献や人の発言を引用するときに使います。
`>` を行頭に付けます。

---

## 基本的な引用

### 記法

```markdown
> これは引用文です。
```

### 表示結果

> これは引用文です。

---

## 複数行の引用

### 記法

```markdown
> これは引用文の1行目です。
> これは引用文の2行目です。
> これは引用文の3行目です。
```

### 表示結果

> これは引用文の1行目です。
> これは引用文の2行目です。
> これは引用文の3行目です。

---

## 段落を含む引用

### 記法

```markdown
> 最初の段落です。
>
> 2番目の段落です。空行も `>` を付けます。
```

### 表示結果

> 最初の段落です。
>
> 2番目の段落です。空行も `>` を付けます。

---

## 入れ子の引用

### 記法

```markdown
> 外側の引用です。
>
>> 内側の引用です。
>>
>>> さらに深い引用です。
```

### 表示結果

> 外側の引用です。
>
>> 内側の引用です。
>>
>>> さらに深い引用です。

---

## 引用内での他の記法

引用内でも他のMarkdown記法が使えます。

### 記法

```markdown
> ## 引用内の見出し
>
> これは**太字**で、これは*斜体*です。
>
> - リスト項目1
> - リスト項目2
>
> ```javascript
> console.log("コードブロックも使えます");
> ```
```

### 表示結果

> ## 引用内の見出し
>
> これは**太字**で、これは*斜体*です。
>
> - リスト項目1
> - リスト項目2
>
> ```javascript
> console.log("コードブロックも使えます");
> ```

## 実用例

### 名言の引用

```markdown
> 私がこれまでくじけずにやってこれたのは、
> ただひとつ、自分のやっている仕事が好きだという、
> その気持ちがあったからです。
>
> — スティーブ・ジョブズ
```

### 表示結果

> 私がこれまでくじけずにやってこれたのは、
> ただひとつ、自分のやっている仕事が好きだという、
> その気持ちがあったからです。
>
> — スティーブ・ジョブズ
