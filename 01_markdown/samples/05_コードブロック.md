# コードブロック（Code Blocks）サンプル

## 解説

プログラムのコードを表示するための記法です。

| 種類 | 記法 | 用途 |
|------|------|------|
| インラインコード | `` `code` `` | 文中の短いコード |
| コードブロック | ` ``` ` で囲む | 複数行のコード |

---

## インラインコード

### 記法

```markdown
`console.log()` は出力に使います。
ファイル名は `index.html` です。
`Ctrl + C` でコピーできます。
```

### 表示結果

`console.log()` は出力に使います。
ファイル名は `index.html` です。
`Ctrl + C` でコピーできます。

## コードブロック（言語指定あり）

言語を指定すると**シンタックスハイライト**（色分け）されます。

### JavaScript

````markdown
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World");
```
````

### 表示結果

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World");
```

### Python

````markdown
```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 出力: 120
```
````

### 表示結果

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 出力: 120
```
