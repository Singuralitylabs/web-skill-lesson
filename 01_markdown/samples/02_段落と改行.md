# 段落と改行（Paragraphs & Line Breaks）サンプル

## 解説

Markdownでは、単にEnterキーを押しただけでは改行されません。

- **段落**：空行（2回Enter）で区切る
- **改行**：行末に半角スペース2つ、または `<br>` を使う

---

## 段落の記法

```markdown
これは最初の段落です。
一つの段落内での改行は無視されます。

これは2番目の段落です。
空行を入れると新しい段落になります。
```

### 表示結果

これは最初の段落です。
一つの段落内での改行は無視されます。

これは2番目の段落です。
空行を入れると新しい段落になります。

---

## 改行の記法

### 方法1：行末スペース2つ

```markdown
1行目
2行目（1行目の末尾にスペース2つ）
```

### 表示結果

1行目
2行目（1行目の末尾にスペース2つ）

### 方法2：brタグ

```markdown
1行目<br>
2行目（brタグを使用）
```

### 表示結果

1行目<br>
2行目（brタグを使用）

---

## 比較表

| 操作 | 結果 |
|------|------|
| Enterキー1回 | 改行されない（同じ段落） |
| Enterキー2回（空行） | 新しい段落 |
| 行末スペース2つ | 段落内で改行 |
| `<br>` タグ | 段落内で改行 |
