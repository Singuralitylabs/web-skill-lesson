# コメント（非表示テキスト）サンプル

## 解説

レンダリングされない隠しメモを文書内に残せます。
自分用のメモや、一時的に非表示にしたい内容に使用します。

**対応環境**: ほぼすべてのMarkdown環境

---

## HTMLコメント

最も一般的な方法です。

### 記法

```markdown
通常のテキスト

<!-- これはコメントです。表示されません -->

この間にコメントがあります。
```

### 表示結果

通常のテキスト

<!-- これはコメントです。表示されません -->

この間にコメントがあります。

---

## 複数行コメント

### 記法

```markdown
<!--
これは複数行の
コメントです。
どれだけ長くても表示されません。
-->

通常のテキスト
```

### 表示結果

<!--
これは複数行の
コメントです。
どれだけ長くても表示されません。
-->

通常のテキスト

---

## 参照形式のコメント

一部の環境で使える別の書き方です。

### 記法

```markdown
[//]: # (これはコメントです)

[//]: # "これもコメントです"

[comment]: <> (括弧形式のコメント)
```

### 表示結果

[//]: # (これはコメントです)

[//]: # "これもコメントです"

[comment]: <> (括弧形式のコメント)

## 実用例

### TODOメモ

```markdown
## 機能一覧

- ログイン機能
- ユーザー登録
<!-- TODO: パスワードリセット機能を追加予定 -->
- プロフィール編集
```

---

### 下書き・一時非表示

```markdown
## リリースノート

### v2.0.0
- 新機能Aを追加
- バグ修正

<!--
### v2.1.0（予定）
- 新機能Bを追加
- パフォーマンス改善
まだ公開しない
-->
```

---

### レビュー用メモ

```markdown
## 実装方針

<!--
レビュアーへ：
この実装はパフォーマンスを優先しています。
可読性とのトレードオフについてフィードバックをお願いします。
-->

以下の方針で実装を進めます...
```

```markdown
# ドキュメントタイトル

<!--
著者：山田太郎
最終更新：2025-01-30
次回更新予定：2025-02-15
-->

## はじめに

このドキュメントでは...
```

### セクションの区切り説明

```markdown
<!-- ========== ヘッダーセクション開始 ========== -->

# タイトル

ヘッダーの内容

<!-- ========== ヘッダーセクション終了 ========== -->

<!-- ========== メインコンテンツ開始 ========== -->

## 本文

メインの内容

<!-- ========== メインコンテンツ終了 ========== -->
```

---

### デバッグ用の一時コメントアウト

```markdown
## 設定手順

1. ファイルをダウンロード
2. 設定ファイルを編集
<!-- 3. 古い手順（削除予定） -->
3. アプリケーションを再起動
```

---

### 注釈・補足情報

```markdown
料金は月額1,000円です。
<!-- 2025年4月から1,200円に値上げ予定。更新時に修正すること -->
```

## コメントの比較

| 形式 | 記法 | 対応状況 |
|------|------|----------|
| HTMLコメント | `<!-- -->` | ほぼすべて |
| 参照形式 | `[//]: #` | 一部 |
| 空リンク形式 | `[comment]: <>` | 一部 |

**推奨**: HTMLコメント `<!-- -->` が最も汎用的

---

## 注意点

1. **ソースには残る**: コメントは表示されないが、ソースコードを見れば読める
2. **機密情報NG**: パスワードやAPIキーをコメントに書かない
3. **HTMLタグとの競合**: コメント内に `--` を含むと問題が起きることがある

### 悪い例

```markdown
<!-- パスワード: secret123 --> ← 絶対にダメ！
<!-- コメント -- 続き --> ← `--` が問題になる可能性
```

### 良い例

```markdown
<!-- TODO: 認証機能を実装する -->
<!-- 注: この値は環境変数から取得すること -->
```

---

## コメントを使うべき場面

| 場面 | 例 |
|------|-----|
| TODO管理 | `<!-- TODO: 〇〇を追加 -->` |
| 一時非表示 | `<!-- 公開前なので非表示 -->` |
| 構造の説明 | `<!-- セクション開始 -->` |
| レビュー依頼 | `<!-- 要確認：この実装でOK？ -->` |
| 更新メモ | `<!-- 最終更新：2025-01-30 -->` |

---

## 実習

以下のドキュメントにコメントを追加してみましょう。

```markdown
# プロジェクト名

## 概要

<!-- TODO: 概要を書く -->

## インストール

1. リポジトリをクローン
2. 依存関係をインストール
<!-- 3. 旧手順（削除予定） -->
3. 環境変数を設定

## 使い方

<!--
レビュアーへ：
使い方のセクションを追加予定です。
-->
```
