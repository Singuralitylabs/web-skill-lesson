# 絵文字ショートコード サンプル

## 解説

絵文字をテキストのショートコードで入力できます。
`:name:` の形式で記述します。

**対応環境**: GitHub、Slack、Discord

---

## 基本的な使い方

### 記法

```markdown
:smile: :thumbsup: :rocket: :warning: :white_check_mark:
```

### 表示結果（GitHub上）

:smile: :thumbsup: :rocket: :warning: :white_check_mark:

## 実用例

### コミットメッセージ

```markdown
:sparkles: 新機能：ユーザー認証を追加
:bug: バグ修正：ログインエラーを解消
:memo: ドキュメント：READMEを更新
:art: UI改善：ボタンのデザインを変更
:zap: パフォーマンス：クエリを最適化
:lock: セキュリティ：XSS対策を実装
:fire: 削除：未使用コードを削除
:wrench: 設定：ESLint設定を更新
```

---

### プルリクエスト

```markdown
## :rocket: 概要

ユーザー登録機能を実装しました。

## :memo: 変更内容

- :sparkles: 新規登録フォームを追加
- :white_check_mark: バリデーション処理を実装
- :art: フォームのスタイリング

## :warning: 注意点

- データベースのマイグレーションが必要です

## :eyes: レビューポイント

- バリデーションのロジック
- エラーハンドリング
```

---

### Issue

```markdown
## :bug: バグ報告

### :computer: 環境
- OS: macOS 14.0
- ブラウザ: Chrome 120

### :footprints: 再現手順
1. ログインページにアクセス
2. メールアドレスを入力
3. :x: エラーが発生

### :bulb: 期待する動作
:white_check_mark: 正常にログインできる

### :camera: スクリーンショット
（画像を添付）
```

```markdown
## :bug: バグ修正

- 画像アップロードの不具合を修正

- ページ読み込み速度を50%改善

## :warning: 破壊的変更
- Node.js 16のサポートを終了
```

---

## 注意点

1. **環境依存**: VS Codeなど一部の環境では`:smile:`がそのまま表示される
2. **代替入力**: 対応していない環境では直接絵文字を入力（😄）
3. **検索方法**: GitHubでは `:` を入力すると候補が表示される
