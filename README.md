## 01_markdown - Markdown記法

Markdown記法を基礎から応用まで解説する教育コンテンツです。

### 構成

- **解説記事（基礎編）.md** - 11種類の基本記法を解説
- **解説記事（応用編）.md** - 10種類の応用記法を解説
- **samples/** - 各記法のサンプルファイル（21種類）

### 学習内容

**基礎編**
- 見出し、段落と改行、リスト、強調
- コードブロック、リンク、引用、表
- 水平線、チェックリスト、画像

**応用編**
- 折りたたみ、脚注、アラート、数式
- Mermaid、絵文字、コメント、エスケープ
- 定義リスト、目次リンク
